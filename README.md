# NODE VERSION >=14.8.0 AND USE NPM

## Setup

1. look at `setup/env.pdf`
3. `npm ci`


## Command

### Install

```shell
$ npm ci
```

### Env

```shell
$ npm run env
```

### Watch

##### watch when development

```shell
$ npm start
```

##### watch when production

```shell
$ npm run start-pro
```

### Build

##### build when development

```shell
$ npm run build-dev
```

##### build when production

```shell
$ npm run build
```

### Build & Watch

```shell
$ npm run serve
```
